Three different designs

This redesign allows for better comparison, but it is not as "beautiful" or different as the original. It also lacks weekly detail. The goal of my visual redesigns was to create a more effective and interpretable representation of new COVID-19 cases while addressing the weaknesses of the original visualization. Each design emphasizes different aspects of the data: the bar chart highlights precise month-to-month comparisons, the heatmap enhances trends over time, and the mountain histogram makes it easier to see long-term case growth. My final visualization, a bubble chart with a heatmap, was an attempt to combine the best aspects of previous designs into a single representation. The bubble size encodes the number of cases per month, while the heatmap provides an additional layer of density information, making it possible to compare trends across years. I wanted to maintain some visual appeal while prioritizing quantitative clarity, ensuring the data was useful for analysis but still engaging. To create the final visualization, I had to work within the constraints of Excel, as no standard visualization tool directly matched my idea. The data had to be cleaned manually, with negative and null values removed, and only new cases per month retained. I then applied a heatmap to highlight density differences and manually scaled bubble diameters based on case numbers. One limitation I encountered was that the bubble size made smaller changes difficult to notice, and the color variation was subtle due to minimal case fluctuations. This illustrates a key challenge in visualization design: emphasizing certain aspects of data can sometimes obscure finer details. Although this redesign addressed some of my initial critiques such as making the data easier to compare and interpret it also reinforced that no single visualization is perfect. Trade-offs exist between clarity, aesthetics, and complexity, and balancing these elements remains a core challenge in data storytelling.
